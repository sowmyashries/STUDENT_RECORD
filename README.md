# 📊 Student Marks Analyzer (C)

A menu-driven **Student Marks Analyzer** developed using **C Programming**. This project was built to enhance my practical C programming skills by applying concepts such as file handling, string manipulation, searching, loops, and conditional statements to a real-world scenario.

## ✨ Features
- Add student records
- Store records in a text file
- Search student records by ID
- View student details
- Display highest marks in Physics, Chemistry, and Mathematics
- Menu-driven interface

## 🛠️ Concepts Used
- C Programming
- File Handling (`fopen`, `fprintf`, `fgets`)
- String Handling (`sscanf`)
- Arrays
- Loops
- Conditional Statements
- Switch Case

## 🎯 Purpose
This project was developed as part of my journey to strengthen my C programming skills through hands-on practice and real-world problem solving.
